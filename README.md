🎓 VITCalc – GPA & CGPA Calculator
🚀 Overview

VITCalc is a modern web application designed for VIT students to easily calculate GPA, CGPA, and Attendance in seconds. It eliminates the need for spreadsheets and provides a clean, intuitive interface for academic planning.

✨ Features
📊 GPA Calculator
Calculate semester GPA by adding courses, grades, and credits
🎓 CGPA Calculator
Combine previous CGPA with current semester performance
🎯 Target CGPA Planner (Optional)
Plan future semesters to reach your desired CGPA
📅 Attendance Tracker (if implemented)
Monitor attendance percentage easily
⚡ Instant Results
Real-time calculations with smooth UI
